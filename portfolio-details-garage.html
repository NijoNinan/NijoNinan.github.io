GARAGE DATABASE MANAGEMENT

  • Category: Program
  • Programming Language: Java
  • Topic: Object Oriented Programming

Detailed Problem statement

To record the details of all the customer and their vehicle and do the services they require and finally make a bill.

Functionalities/features supported by the project

  • The program stores any number of customers that enter the Garage.
  • It stores all the services done by each customer till the make their final payment.
  • On final payment it displays all the services that you had done and the number of times that you had done the same, and also displays the final amount that you have to pay.
  • You can toggle between customers any time you want.
  • Even if you make payment, the details of the customer and their vehicles are not deleted. Hence If the customer later wants to enter the garage again the they need not enter the whole details again.

Used Concepts

  • Vector
  • Inheritance
  • Abstract Class
