BEAUTIFUL NUMBER CALCULATOR

  • Category: Program
  • Programming Language: Python
  • Topic: Analysis of Algorithms

Detailed Problem statement

The objective of our program is to find the smallest beautiful number of 'n' digits,
where 'n' is the input given by the user.

What is a beautiful number?

Let's call a positive integer 'M' a beautiful number if the following conditions are satisfied:

  • The decimal representation of 'M' does not contain any zeros
  • The sum of squares of all decimal digits of 'M' is a perfect square
For example; 1, 43 and 221 are beautiful, but 12 and 333 are not.
Explanation:
1 → 12 = 1 → perfect square → beautiful number
43 → 42 +32 = 25 → perfect square → beautiful number
221 → 22 + 22 + 12 = 9 → perfect square → beautiful number
12 → 12 + 22 = 5 → non perfect square → not a beautiful number
333 → 32 + 32 + 32 = 27 → non perfect square → not a beautiful number

Functionalities / features supported by project

  • The code is written in python language
  • You can give the input from 0 up to 106 i.e. the program calculates the smallest beautiful number up to 106 digits
  • Graphical user interface (GUI) has been added using tkinter
  • You can give the input in the text box appearing in the window
  • The calculate button calculate the smallest beautiful number of the given number of digits and displays the same
  • Clear button clears the text box and the previous result displayed in the window
  • The exit button exits the program.

Working of the Algorithm:

The user gives the value of ‘n’ where n is the number of digits of the result.
Logically, the smallest non zero number of n digits is a number where all the digits are 1.
So we initially we make a list of n elements where each element is 1; this represents the result
So the sum of squares of all digits is n itself ; since all the digits are 1.
After every time the program checks whether the sum is a perfect square. If yes then the result is the number in the list.
Else we increment the value of the last element of the list by one, update the new sum. If the value of the last element is 9, then we increment the value of the previous element by one also reset the value of present element.
Thus repeat the above step, till the value sum is a perfect square.
The number formed by joining all the elements of the list is the final result
